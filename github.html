Hva er Github?

For å forstå hva Github er, må man først opparbeide seg en forståelse av GIT. Git er et kontrollsystem for filversjoner.

Github er en webbasert "hosting" tjeneste for versjonkontrollering av av filer med bruk av GIT. Github er for det meste brukt for datakode.
